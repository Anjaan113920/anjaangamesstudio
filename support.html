Support

Anjaan Games Studio

If you need help with any of our games, we are happy to assist you.

This support page applies to all games developed by Anjaan Games Studio.

Common Issues

  • Restart the game if something is not responding
  • Make sure your device is running the latest iOS version
  • Update the game to the latest version from the App Store

In-App Purchase Support

  • Purchased content should unlock automatically
  • If content is missing, use the Restore Purchases option in the game
  • All payments are securely processed by the Apple App Store
  • Anjaan Games Studio does not store payment information

Game Issues

If you experience crashes, bugs, or gameplay issues, please contact us and include:

  • Game name
  • Device model (example: iPhone 14)
  • iOS version
  • Description of the issue
